What Gets You a Level 3

  • L1 base: All 6 knowledge areas covered with all tools listed below applied to the project.

  • L3 addition (locked choice): Reflection essay or article for CODE/external community or workshop. Must confirm choice with Swantje Quoos.

L1 tools required: Problem statement, primary research summary (≥5 interviews in appendix), secondary research, user group definition + quantification, 1–3 Personas, user journey map, Value Proposition Canvas, competitor matrix, PESTLE, TAM/SAM/SOM (bottom-up + top-down), market prognosis, beachhead market, solution description, value proposition statement, SWOT, prototype description, ≥3 hypothesis cards, experiment board, prototype test results, MVP + user story map, Build-Measure-Learn cycle.
